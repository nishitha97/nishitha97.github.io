Projects In Detail

Online Mobile Accessories Shop (Year 1 semester 2)

This project was an e-commerce website developed using Php, CSS and JavaScript which allows customers to view the mobile accessories available in the store and purchase them. MySQL database was used with PhpMyAdmin run on Xampp

Online Shopping Store (Year 2 semester 1)

The online shopping store is a dynamic e-commerce web application for a shop operating on a B2C environment which sells Fast Moving Consumer Goods. This project was developed using Java, jsp and servlet technologies, JavaScript and CSS. The online shopping store developed, allows customers who visit the site to view available items in the store and purchase them. This project also implements all the functions necessary for the management and handling of day to day operations of the store including inventory and product management, customer order handling, product delivery handling as well as sales report generation. MySQL database was used with PhpMyAdmin run on Xampp.

Class Management System for IdeaCurl institute (Year 2 semester 2)

The Class Management System for IdeaCurl educational institute was developed using Java, jsp and servlet technologies, JavaScript and Bootstrap. The Class Management System includes all the functions required by the client and is comprised of a student and lecturer registration sub system with generation of an ID card with a unique barcode, student, employee and lecturer attendance management sub system, institute employee management sub system, lecturer class hall allocation handling and schedule management sub system, institute financial management sub system which includes lecturer payment handling, student payment handling as well as an institute inventory management sub system to keep track of the stationery items available at the institute. MySQL database run on SQLWorkbench was used as the database for this project.

Student Lecturer Interactive App (Year 2 semester 2)

This mobile application is a Java based Android application developed using Android Studio for students and lecturers. Students using this application can interact with lecturers in their classes by posting questions to the lecturer, give feedback on the lectures conducted by lecturers and also make an appointment request to meet a lecturer if needed. Lecturers using this application can view the feedback given by students about their lectures, reply to questions asked by students, view appointment requests made by students in the lecturer’s class as well as notify students in the lecturer’s class about any events in the class.

Train Ticket Reservation System (Year 3 semester 1)

This application is a web-based application making use of Spring Boot and ReactJS technologies. The developed application encompassed all features and functionality of a standard train ticket reservation system including the ability to view train time tables and schedules of a particular station, make reservations, select from various types of available payment options and also generate virtual online tickets.

Student and Instructor Information System (Year 3 semester 1)

This application is a simple web-based Student and Instructor information System developed primarily for the user roles of instructor, student and system administrator. The developed system is mainly comprised of client and server components in which NodeJS was used for server-side development and ReactJS for the client-side development. The system includes all the necessary features and functionalities as found in most similar student and instructor information systems such as registering and creation of user accounts, allocation of user roles, file and document upload functionality, email notification service, course and assignment creation as well as key security and authentication features such as user authentication token and password encryption.

Code Complexity Analyzer (Year 3 semester 2)

This project was a standalone application designed to analyze code complexities in Java and C++ applications. The application’s primary purpose was to analyze code complexity of program statements based on four main criteria namely size, inheritance, nesting levels and recursion. Java along with a specially developed ANTLR parser was used for complexity identification purposes and JavaFX was used to develop the user interfaces. Here all the code complexities identified relevant to a particular complexity criterion, are given a pre-defined score, and displayed to the user such that the user can get an idea and understanding on complexity of a particular code file or program statement.

Public Transport Ticketing System (Year 3 semester 2)

This Public Transport Ticketing System developed was mainly developed targeting bus and railway transport and includes all the key features and functionalities required such a system including the functionalities such as the ability to reserve a particular seat, view travel time table information, view passenger journey history as well as a unique QR code based approach when handling passenger seat reservation and payment. Furthermore, administrator-based functions such as the functionality to add, update or delete journey routes, route inspectors as well as time table information were also implemented in the development of this system. A key point to note in the development process of this system is that this system was developed based on design document handed down to the group members and that all user interfaces, features and functionality were developed as per the documentation provided in the design document.

Mobile App for Sri Lanka Cricket (Year 3 semester 2)

This mobile application is a Java based Android application developed using Android Studio. This mobile application was developed in response to solve the User Experience (UX) issues identified in the official app of Sri Lanka cricket. When compared with the existing official mobile application of the Sri Lanka cricket team, the newly developed mobile application was able to improve on key aspects such as the performance, usability and overall design. Furthermore, additional features were also included to the developed application in order to provide the user with a better user experience.
